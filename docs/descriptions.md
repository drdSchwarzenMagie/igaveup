### Тест 1: Загрузка директории с треками
1. Действие: Нажмите кнопку "Load Directory" и выберите директорию с mp3 файлами.
2. Ожидаемый результат:
   - Все доступные mp3 файлы в выбранной директории должны появиться в списке плейлиста.
   - Метка текущего трека должна измениться на "No track loaded" (если ранее не был загружен трек).

### Тест 2: Воспроизведение трека
1. Предварительное условие: Директория с треками должна быть загружена.
2. Действие: Нажмите кнопку "Play".
3. Ожидаемый результат:
   - Должен начаться воспроизводиться первый трек из списка.
   - Метка текущего трека должна измениться на "Now playing: <название трека>".

### Тест 3: Пауза и возобновление воспроизведения
1. Предварительное условие: Трек должен воспроизводиться.
2. Действие: Нажмите кнопку "Pause".
3. Ожидаемый результат: Воспроизведение трека должно приостановиться.
4. Действие: Нажмите кнопку "Pause" снова.
5. Ожидаемый результат: Воспроизведение трека должно возобновиться.

### Тест 4: Остановка воспроизведения
1. Предварительное условие: Трек должен воспроизводиться.
2. Действие: Нажмите кнопку "Stop".
3. Ожидаемый результат: Воспроизведение трека должно остановиться.

### Тест 5: Воспроизведение следующего трека
1. Предварительное условие: Должно быть несколько треков в плейлисте.
2. Действие: Нажмите кнопку "Next".
3. Ожидаемый результат:
   - Должен начать воспроизводиться следующий трек.
   - Метка текущего трека должна измениться на "Now playing: <название следующего трека>".

### Тест 6: Воспроизведение предыдущего трека
1. Предварительное условие: Должно быть несколько треков в плейлисте.
2. Действие: Нажмите кнопку "Previous".
3. Ожидаемый результат:
   - Должен начать воспроизводиться предыдущий трек.
   - Метка текущего трека должна измениться на "Now playing: <название предыдущего трека>".

### Тест 7: Выбор трека из плейлиста
1. Предварительное условие: Директория с треками должна быть загружена.
2. Действие: Выберите трек из списка плейлиста.
3. Ожидаемый результат:
   - Должен начаться воспроизводиться выбранный трек.
   - Метка текущего трека должна измениться на "Now playing: <название выбранного трека>".

### Тест 8: Включение и выключение звука
1. Предварительное условие: Трек должен воспроизводиться.
2. Действие: Нажмите кнопку "Mute/Unmute".
3. Ожидаемый результат: Звук трека должен выключиться.
4. Действие: Нажмите кнопку "Mute/Unmute" снова.
5. Ожидаемый результат: Звук трека должен включиться.

### Тест 9: Пустая директория или отсутствие mp3 файлов
1. Действие: Нажмите кнопку "Load Directory" и выберите пустую директорию или директорию без mp3 файлов.
2. Ожидаемый результат: 
   - Плейлист должен оставаться пустым.
   - Метка текущего трека должна оставаться на "No track loaded".
